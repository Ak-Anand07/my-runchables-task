## MySQL Setup

Auth storage uses MySQL.

1. Copy `.env.example` to `.env.local` and set your MySQL credentials.
2. Install dependencies:

```bash
npm install
```

3. Run the app:

```bash
npm run dev
```

The app creates the `users` table automatically when auth APIs are called.
